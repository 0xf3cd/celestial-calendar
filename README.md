# Celestial Calendar
> A C++20/23-style calendar

## Features
* Conversions between gregorian, lunar, and ganzhi dates (公历、阴历、干支历之间的转换)
* Accurate Jieqi moments queries (查询某一年的某节气的具体时刻)
